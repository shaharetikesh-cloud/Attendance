सुरुवात

हे software attendance सोपे ठेवण्यासाठी तयार केले आहे. Substation save करा, employee add करा, मग month निवडून attendance generate करा.

Employee master

  • Joining date optional आहे.
  • Working place outsource किंवा special duty साठी वापरा.
  • Operator, Tech/Engineer, Apprentice, Outsource, Other असे groups आहेत.

Attendance logic

  • P present
  • A absent
  • - blank / outside duty / joining date आधीचा दिवस
  • CL, EL, HCL, Medical, OD, LWP, C-OFF codes उपलब्ध आहेत.

कामाची पद्धत

  1. Substation निवडा.
  2. Month आणि year निवडा.
  3. Generate दाबा.
  4. फक्त exception बदल करा.
  5. Save करा. तो month नंतर पुन्हा open होतो.
  6. Export PDF ने print-ready कागद मिळतो.

Outsource

Outsource attendance PDF मध्ये heading वेगळी ठेवलेली आहे. Working place row level वर दाखवता येते.

टीप

Manual remark box मध्ये बाहेर training, deputation, special duty, किंवा office note लिहा.
